Python Calculator

This program is a simple calculator built using Python's Tkinter library for the graphical user interface (GUI). It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The interface includes a text entry field for displaying input and results, and buttons for each digit (0-9) and operation. Users can input numbers and perform calculations by clicking the corresponding buttons. The program supports functionality to clear the display, calculate the result of an operation, and handle the basic arithmetic logic. It also features global variables to store the first number entered and the current operation, ensuring accurate calculation when the equal button is pressed.
